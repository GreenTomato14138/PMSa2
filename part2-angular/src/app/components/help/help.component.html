Help & FAQs

How to use the application?

Use the navigation menu to access different features:

  • Home: View summary and popular items.
  • Manage Items: Add, update, or delete inventory items.
  • Search: Search for items by name, view all items, or popular items.
  • Privacy: Learn about security measures.

Troubleshooting

Q: I added an item but it disappears after refresh.
A: This app stores data only in memory during the session. Data is lost on refresh by design.

Q: Can I edit an item by name?
A: Yes, in the Manage Items page, enter the current name and provide the fields you want to update.

Q: Why can't I add duplicate item names?
A: Item names must be unique to ensure accurate identification. Use the update feature to modify existing items.

Q: I see an error about negative quantity/price.
A: Quantity and price must be zero or positive numbers.

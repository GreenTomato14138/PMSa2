# Part 1: TypeScript Inventory Management System

## Features
- Add, edit, delete items
- Search by name
- View all items and popular items
- Responsive design
- No external libraries

## How to Run
1. Open `index.html` in a modern browser (with TypeScript compiled to JS).
2. Ensure `inventory.js` is generated from `inventory.ts`.
3. All data is stored in memory during the session.

## Note
The TypeScript file must be compiled to JavaScript. Use `tsc inventory.ts` to generate `inventory.js`.

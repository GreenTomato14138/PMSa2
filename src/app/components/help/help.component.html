❄️ Grimoire of Guidance

📜 How to forge a new artifact?

Enter the Vault → fill the creation form (Sigil, True Name, Supplier are mandatory) → click “Conjure”. Sigil must be unique across the realm.

✧ How to alter an existing entry?

Inside the Grimoire, press “Edit” on any card. Modify any field except the Sigil (immutable), then “Update Essence”.

✖ How to banish an artifact?

Click “Banish” on the product card → confirm the vanishing. This action is irreversible and echoes into the void.

🔮 How to search across realms?

Go to the Oracle page. Enter a name, filter by category and price range, then “Scry” to reveal matches.

✨ What makes an item “Radiant”?

Radiant items are manually flagged in the Vault form. They appear with a ✦ badge and can be filtered via the “Show Radiant Only” button.

💾 Is data saved permanently?

This is a ephemeral demonstration. All data lives in memory only. Refreshing the page restores the default seed data. For persistent storage, integrate a backend.

⚠️ What if the ritual fails?

Check all mandatory fields, ensure numeric values are valid, and Sigil is unique. For deeper issues, consult the Angular scriptures.

🛠️ Runic Troubleshooting

  • Conjuration rejected: All * fields must be filled (Sigil, True Name, Supplier).
  • Sigil conflict: Choose a different unique Sigil.
  • Oracle returns nothing: Broaden your incantation or reset filters.
  • Sigil cannot be changed during edit: Sigils are immutable anchors; delete and re‑forge if needed.
  • Visual glitches? Clear cache or use latest Chrome/Edge.
